# Applying Modern Techniques to Action Classification THUMOS14

| Step | Pipeline Stage                      | UvA THUMOS 2014 Approach                                     | Modern Approach to Try                                              | Notes                                                               |
| ---- | ----------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------------- | ------------------------------------------------------------------- |
| 1    | **Video Input**                     | Raw video                                                    | Raw video                                                           | Same starting point                                                 |
| 2    | **Frame / Clip Sampling**           | Dense frame processing for trajectory extraction             | Uniform sparse clip sampling (8–32 frames per clip)                 | Deep models do not require dense frame processing                   |
| 3    | **Low-Level Feature Extraction**    | Improved Dense Trajectories (HOG, HOF, MBH descriptors)      | Video transformer backbone (VideoMAE / VideoMAEv2)                  | Deep models learn spatial and motion features jointly               |
| 4    | **Appearance Feature Extraction**   | ImageNet CNN applied per frame (object-centric CNN features) | Already integrated inside video backbone                            | Modern models unify motion + appearance                             |
| 5    | **Feature Encoding**                | Fisher Vector encoding of trajectory descriptors             | Direct embedding from backbone                                      | Fisher vectors compress descriptors into global features            |
| 6    | **Temporal Modeling**               | Mostly none (features aggregated across video)               | Temporal attention or clip-level sequence modeling                  | Transformer attention captures long-range motion patterns           |
| 7    | **Feature Aggregation**             | Global pooling of encoded features                           | Temporal pooling across clip embeddings (mean or attention pooling) | Determines how multiple clips combine into one video representation |
| 8    | **Classifier Input Representation** | Concatenated Fisher vectors + CNN features                   | Single video embedding vector                                       | Deep features replace multiple handcrafted streams                  |
| 9    | **Classifier**                      | One-vs-rest linear SVM                                       | Linear softmax classification head                                  | Enables end-to-end training                                         |
| 10   | **Training Strategy**               | Train SVM on fixed features                                  | Fine-tune pretrained video backbone                                 | Backbone adapts to dataset                                          |
| 11   | **Inference Strategy**              | Single representation per video                              | Multi-clip inference (average predictions across clips)             | Improves robustness                                                 |
